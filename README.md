## 프로젝트 개요

- 프로젝트명
  - PKKK 플레이스

- 목적
  - 사용자 위치를 기반으로 주변 상점의 리뷰 데이터 제공

- 기능
  - 회원가입 및 로그인, 개인정보 편집 기능
  - 장소 리뷰 열람 및 작성, 편집 기능

- 요구사항
  - 데이터베이스 테이블 작성 및 서버 구축
  - 위치 기반 서비스를 위한 Google API 발급
  - 사용자 인터페이스 설계 및 구축
